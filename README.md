# Pucora circuit breakers

A collection of circuit breaker adaptors for [Pucora](https://github.com/pucora/velonetics-ce)

### gobreaker

The package `gobreaker` contains an adaptor over the github.com/sony/gobreaker lib

### eapache

This is a simple proxy adaptor for the github.com/eapache/go-resiliency/breaker circuit breaker implementation

## Documentation

1. Official: https://pucora.io/docs/throttling/circuit-breaker/

2. Autogenerated: https://godoc.org/github.com/pucora/velonetics-circuitbreaker
